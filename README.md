Simple Calculator

Simple Calculator is a lightweight web application that performs basic arithmetic operations like addition, subtraction, multiplication, and division. It provides a clean interface and instant results, making it ideal for beginners and quick calculations.

Simple Calculator — Prototype

A frontend-only calculator application built using HTML, CSS, and JavaScript.
It allows users to perform basic calculations directly in the browser without any backend or server setup.
Data is not stored permanently and resets on page refresh.

Features
Basic arithmetic operations (+, −, ×, ÷)
Real-time input display
Instant result calculation
Clear screen functionality (C button)
Delete last input (Backspace ⌫)
Decimal number support
Simple and user-friendly interface

How It Works
User inputs numbers and operators through buttons
Expression is displayed on the screen
JavaScript processes the expression using eval()
Result is calculated and shown instantly

Technologies Used
HTML5
CSS3
Vanilla JavaScript


Notes
Works entirely in the browser
No data is stored or saved
Uses eval() for calculation (safe for learning, not recommended for production)


Limitations
Supports only basic arithmetic operations
No scientific or advanced functions
No keyboard input support
Not suitable for large-scale or secure application
